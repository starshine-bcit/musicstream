Dependencies:

eyeD3
python-magic-bin (windows)

To Install:
    pip install eyeD3
    pip install python-magic-bin

streamserv invocation:

    help:
        py streamserv.py help

    create the database:
        py streamserv.py db create your/path/here
            Note: path must be absolute

    db stats:
        py streamserv.py db stats
